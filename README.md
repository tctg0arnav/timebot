# timebot

Invite - https://arnavshukla.co/timebot

# how to use
!time {timezone}
#to host your own
run the python script. set your env variables as follows:

DISCORD_TOKEN={discord_bot_token_here}
use the discord OAUTH manager to invite the bot to your server
